## Installation

```bash
cd reactive_fabric_analysis
uv sync
```

## Running

```bash
uvicorn main:app --reload --port 8000
```

## API Endpoints

- `GET /` - Root endpoint
- `GET /health` - Health check

## Testing

```bash
pytest tests/
```

## Configuration

Environment variables:
- `PORT` - Service port (default: 8000)
- `LOG_LEVEL` - Logging level (default: INFO)
